FILES         top

    llvm-project-widberg-extenstions - source code
    Clang/LLVM fork with first-class __usercall syntax support.

    fmtk - source code
    Mod injector/loader for Asobo's FUEL (2009) video game.

    usercall.hpp - source code
    MSVC++ macro library for generating function prologues/epilogues for
    an IDA style usercall/userpurge function declaration.

    dpc - source code
    Tar analogous utility to work with the DPC file format from Asobo Studio games.

    fuel-map - source code
    FUEL map explorer/annotator using the Google Maps JS Api and GIS data.

    fuel-brute - source code
    CRC32 Brute Force Solver For Asobo Studio file path hashes using CUDA.

    kasm - source code
    A MIPS-like virtual machine, compiler, and assembler.

    kyte - source code
    Shader language and compiler.

    tmpl - source code
    CLI template replacement tool.

    bgfx.cmake/bigg - source code
    Independently maintained CMake build scripts for bgfx. Released under
    public domain. Starter project for bgfx, imgui, glfw, glm.

    pcil - source code
    A python CLI tool used to generate a single header C++ library for
    getting PCI device names.
